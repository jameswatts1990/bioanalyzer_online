Femto Trace Workflow

bioanalyzeR Online

Run Femto Trace differential analysis entirely in-browser and review R-style output, summary KPIs, and visualizations in one place.

Getting Started (New Users)

  1. Run the app: start this UI with python3 -m http.server 8080, then open http://localhost:8080.
  2. Add Femto Trace data: paste CSV/TSV where each row is a marker/gene and columns are sample intensity values.
  3. Choose sample columns: pick which samples belong to Group A and Group B using the selectors below.
  4. Label sample groups: set Group A / Group B to your comparison (for example Control vs Treated).
  5. Run analysis: click Run Femto Trace Analysis to calculate fold-change and adjusted p-values locally.
  6. Review results: use KPI cards, an R output panel, volcano visualization, and the results table.

Tip: Use Load Demo Payload for a ready-to-run local example.
