Femto Trace Workflow

bioanalyzeR Online

Start with a Femto Trace sample table, connect to your backend, and review differential analysis results in one place.

Getting Started (New Users)

  1. Run the app and backend: start this UI with python3 -m http.server 8080, then open http://localhost:8080.
  2. Set API Base URL: enter your bioanalyzer backend URL and click Test Connection.
  3. Add Femto Trace data: paste CSV/TSV where each row is a marker/gene and columns are sample intensity values.
  4. Label sample groups: set Group A / Group B to your comparison (for example Control vs Treated).
  5. Run analysis: click Run Femto Trace Analysis to calculate fold-change and adjusted p-values.
  6. Review results: use KPI cards for a quick summary and the table for per-marker interpretation.

Tip: Use Load Demo Payload for a ready-to-run example.
