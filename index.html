Femto Trace Workflow

bioanalyzeR Online

Analyze Femto Pulse / ProSize electropherogram exports in-browser with clear setup guidance, QC-focused thresholds, and visual review in one place.

Getting Started (New Users)

  1. Run the app: start this UI with python3 -m http.server 8080, then open http://localhost:8080.
  2. Export from ProSize (Femto Pulse): export Electropherogram (single file, time scale), Peak Table (alternate), and Size Calibration CSVs from the same run folder.
  3. Add Femto Trace matrix data: paste CSV/TSV where each row is a region/feature and each sample column contains intensity, concentration, or molarity values.
  4. Detect and verify sample columns: click Detect Sample Columns, then confirm Baseline and Test sample assignments.
  5. Choose labels and thresholds: set Baseline/Test names and QC thresholds for adjusted p-value and |log2FC|.
  6. Run Femto Trace analysis: click Run Femto Trace Analysis to calculate per-feature fold changes and adjusted p-values locally.
  7. Review results: use KPI cards, a bioanalyzeR-style console preview, volcano visualization, and the results table.

Tip: Use Load Demo Payload to load a Femto-style 3-vs-3 starter dataset, detect columns automatically, then run analysis in one click.
