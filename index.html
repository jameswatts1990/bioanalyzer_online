Femto Trace Workflow

bioanalyzeR Online

Analyze Femto Pulse / ProSize electropherogram exports in-browser with clear setup guidance, QC-focused thresholds, and visual review in one place.

Getting Started (New Users)

  1. Run the app: start this UI with python3 -m http.server 8080, then open http://localhost:8080.
  2. Preview a working example first: click Load Demo Payload to see a valid starter Femto CSV shape and expected sample grouping behavior.
  3. Prepare your Femto CSV: row 1 is headers, column 1 is feature ID, and each remaining column is one numeric sample replicate.
  4. Name sample columns clearly: include group labels in headers (for example, control_rep_1 and treated_rep_1) to make assignment easier.
  5. Paste CSV/TSV data: add the matrix to Femto Trace Data Matrix and verify there are no blank headers or non-numeric values in sample columns.
  6. Detect and verify columns: click Detect Sample Columns, then confirm Baseline and Test sample assignments in both selectors.
  7. Set labels and thresholds: choose Baseline/Test names and QC cutoffs for adjusted p-value and |log2FC|.
  8. Run and review: click Run Femto Trace Analysis, then review KPI cards, console preview, volcano visualization, and result table.

Tip: First-time users should run the demo once, then replace only the matrix with their own Femto CSV while keeping the same format.
